## 1. 제품 개요
설치·가입 없이 브라우저에서 동작하는 신혼부부 공용 가계부. 수입·지출·저축·투자·대출·장기목표·생활비통장·일정·메모를 한 앱에서 관리하고, PC↔모바일을 클라우드로 동기화한다. 데이터는 기본적으로 기기 `localStorage`에 저장되고 JSON/CSV로 내보낼 수 있다.

## 2. 사용자 & 페르소나
- **은정 / 성현 (신혼부부)**: 함께 목표(집·육아·목돈)를 세우고 매달 가계를 기록·공유.

## 3. 정보구조 (메뉴)
대시보드(상단 로고 클릭) · 월별 가계부 · 예산 설정 · 수입 관리 · 지출 관리 · 생활비 통장 · 저축 관리 · 주식 투자 · 대출 상환 · 장기 프로젝트 · 일정 공유 · 백업/복원 · 설정
> 메뉴 이름·순서는 설정에서 사용자가 변경(드래그/직접 입력) 가능.

## 4. 기능 요구사항 (Functional Requirements)

### 4.1 대시보드
- 위젯: 핵심통계(수입·지출·저축·남은돈), 지출구성·저축률, 카테고리별 지출, 결제수단별 비율, 대출 잔액, 장기 프로젝트, 다가오는 일정, 우리집 메모판(달력+메모), 최근 입력.
- **편집 모드**: 위젯 표시/숨김 체크, **드래그/▲▼로 순서 변경**(자동 저장).
- 카드 클릭 시 해당 페이지로 이동(편집 모드에서는 비활성).
- **남은 돈(이월 포함)** = 시작잔액 + 누적수입 − 누적지출(결제완료) − 순저축(입금−인출, 이월 제외). 전월 이월 자동 누적.
- **수입 표시 기준**: 이번 달/지난 달 토글(말일 월급 대응, 기본 '지난 달').
- 우리집 메모판: **왼쪽 달력 2/3 + 오른쪽 메모 1/3**, 달력 날짜 클릭 시 일정 추가 이동.

### 4.2 수입 관리
- 항목: 날짜, 사람(은정/성현), 종류(월급/상여/부수입/기타), 금액, 메모.
- 월별 합계·사람별 합계. 모달 수정/삭제.

### 4.3 지출 관리
- 항목: 날짜, 구분(공동/아내 개별/남편 개별), 카테고리, 결제수단, 금액, 메모.
- 카테고리: 식비·교통·통신·보험·병원·쇼핑·경조사·데이트·가전가구·여행·**대출상환·저축**·카드값·현금·기타.
  - **대출상환** → 세부(원금/이자) + 대상 대출 선택. 원금이면 **대출 잔액 자동 차감**(삭제 시 복구).
  - **저축** → 목적 선택. 해당 목적 **저축 목표 진행률에 자동 합산**.
- 결제수단: 생활비통장·**신용카드·체크카드**·현금·계좌이체·기타.
- **신용카드 결제 예정**: 신용카드 지출은 즉시 지출로 안 잡히고 "결제 예정"으로 분리(공동/은정/성현별 집계). **결제일에 '결제완료'** 누르면 그날 지출로 확정.
- **고정 지출**: 매월 반복 항목을 미리 등록(매월 며칠) → 해당 월 '지출 완료' 체크 시 실제 지출로 기록.
- 금액 입력 시 천 단위 콤마 자동, 모달 수정/삭제.

### 4.4 생활비 통장
- 생활비를 통장에 입금(이체)하고, 결제수단 '생활비통장' 지출로 차감 → **잔액 자동 계산**(입금 합계 − 통장 지출).

### 4.5 저축 관리
- 항목: 날짜, **유형(모으기 입금/찾기 인출)**, 목적, 금액, 목표금액, **만기일(적금)**, 메모, **기존(이월)** 체크.
- 입금 = 현금 감소, 찾기 = 현금 증가(남은 돈에 반영). 이월은 현금 제외·목표 누적만.
- 목적별 진행률(저축 관리 + 지출관리 '저축' 통합), 만기일 D-day.

### 4.6 주식 투자
- 항목: 종목명, 종류(국내/해외주식·ETF·펀드·채권·발행어음·CMA·파킹·코인·기타), 명의, 투자원금, 평가금액, **만기일**, 메모.
- 평가손익·수익률 자동, 추가 매수/평가액 갱신.
- **만기 도래 시 자동 수입 전환**(발행어음 등): 만기일 지나면 평가금액을 수입으로 기록하고 자산 합계에서 제외.

### 4.7 대출 상환
- 항목: 대출명, 최초/현재 잔액, 금리, 월 상환액, 상환일, 남은 개월수, 메모.
- **월 납입액 자동 계산**: 잔액·금리(·개월수)로 이자만/원리금균등 계산.
- 월별 상환 내역(원금/이자 분리 입력) → 원금만큼 잔액 차감.

### 4.8 장기 프로젝트
- 항목: 프로젝트명, 목표금액, 현재금액, 목표일, 우선순위, 상태.
- 진행률, 목표일까지 기간, **월 적립 필요액 자동 계산**, 완료 처리.

### 4.9 일정 공유
- 분류: **재정 / 개인 / 근무 / 가족** (분류별 카테고리, 근무=연차·반차·대체휴무·당직·출장·교육·회식·출근).
- 개인·근무 일정은 **사람(은정/성현)** 지정.
- **반복**: 한 번 / 매주·격주(요일 선택) / 매월(며칠·말일) / **매년(양력·음력)** — 음력은 Intl 단기 달력으로 변환.
- 시작/종료일(기간), 시작/종료 시간(미선택 시 하루 종일).
- **월 달력**(공휴일·주말 색상, 오늘 강조), 다가오는 일정 D-day, **공유 텍스트 복사**(카톡 등).
- 분류 필터, 모달 수정/삭제.

### 4.10 우리집 메모판
- 메모: 작성자(은정/성현), 날짜, 본문.
- **캐릭터 이모지 반응**(누가 눌렀는지 표시, 재클릭 시 취소), **댓글**(작성자 선택)과 댓글 반응.
- 메모 **개별 보관(숨김)** + **보관함 리스트**(꺼내기/삭제).

### 4.11 예산 설정
- 월별: 공동생활비·아내·남편 예산, 저축 목표액·목표 저축률. 예산 대비 사용 초과/절약 표시.

### 4.12 데이터 관리
- JSON 백업/복원, CSV(엑셀, BOM) 내보내기, 샘플만 삭제, 전체 초기화(확인창).
- 시작 잔액 설정, 이름(표시용), PIN 변경, 메뉴 편집.

### 4.13 클라우드 동기화 (Supabase)
- 입력 시 자동 업로드(디바운스), 앱 복귀/실시간 구독 시 자동 수신.
- 수동 올리기/받기(받기는 강제 덮어쓰기), 상태 표시.
- **설정 링크 한 번**으로 다른 기기 자동 연결(`#setup=...`), 코드 붙여넣기 지원.

## 5. 비기능 요구사항 (Non-functional)
- **단일 HTML**: 외부 프레임워크 없이 인라인 CSS/JS. (동기화용 supabase-js만 CDN)
- **반응형**: 모바일(소형 폰 포함)·PC. 가로 넘침 방지, 입력칸 1열 정리.
- **PWA**: 매니페스트 + 서비스워커(network-first) + 앱 아이콘, 홈 화면 설치/오프라인.
- **보안/프라이버시**: 금융 데이터는 기기·사용자 Supabase에만 저장. 코드/저장소에 비밀정보 미포함. PIN 잠금.
- **UX**: 따뜻한 신혼집 감성(베이지·크림·브라운·코랄), 카드형·둥근 모서리·부드러운 그림자, 원화·천단위 콤마, 빈 상태 안내.
- **호환/접근**: 마우스 뒤로가기·모바일 왼쪽 가장자리 스와이프 뒤로가기, 키보드 PIN 입력.

## 7. 기술 스택 / 배포
- HTML·CSS·JavaScript(바닐라), `localStorage`, Intl(단기 달력), PWA(sw.js, manifest.json).
- 동기화: Supabase(`households` 테이블 1행에 전체 JSON 저장, 실시간 구독).
- 호스팅: GitHub Pages (`EJ20179/honeybudget`, main 브랜치).

## 8. 주요 사용자 시나리오 (User Stories)
- US-1: 말일에 월급을 받으니 대시보드는 '지난 달' 수입 기준으로 본다.
- US-2: 신용카드로 쓴 건 결제 예정에 모였다가 결제일에 '결제완료'로 그날 지출에 반영한다.
- US-3: 적금에 만기일을 넣고, 만기에 '찾기'로 현금에 돌려받는다.
- US-4: 지출 관리에서 '대출상환·원금'으로 적으면 대출 잔액이 자동으로 준다.
- US-5: 연차·당직을 근무 일정으로 등록하고 달력에서 부부가 함께 본다.
- US-6: 설정 링크를 배우자에게 보내면 그 기기에서 링크만 열어 동기화된다.

## 9. 향후 과제 (Backlog)
- 총자산 현황(현금+저축+투자−대출=순자산) 통합 화면
- 실시간 시세·환율 연동(선택), 카테고리 사용자 정의
- 음력 미지원 환경 대비 내장 변환표, 일정 알림(푸시)

## 10. 변경 이력 (요약)
초기 단일 HTML 가계부 → PIN/반응형 → 생활비통장·투자·일정 → 대시보드 편집/드래그 →
PWA·GitHub Pages 배포 → Supabase 동기화/설정 링크 → 신용카드 결제예정·고정지출 →
현금흐름 정합(시작잔액·이월·저축 입출금·대출/저축 지출 연동) → 일정 반복 확장(주/격주/년·음력)·달력 → 메모 반응/댓글/보관함.
